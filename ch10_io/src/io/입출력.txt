입출력(Input/Output)

- 기본입력 : 키보드
- 기본출력 : 화면

- System.out(화면)
- System.in(키보드)
   new Scanner(System.in)

- 입력장치(키보드, 파일 ,객체...등)에서 읽어오기
 1) text 파일 : 문자 스트림
 2) text + 모든파일 : 바이트 스트림

- 스트림(Stream) : 서로 연결된 두 대상의 연결통로
 - 스트림은 양방향 사용 불가
 - 입력, 출력을 따로 처리해야 함

- File : 자바에서 파일 처리를 담당하는 객체
  - 윈도우에서 사용하는 경로 구분자 : ;(세미콜론)
  - 유닉스(리눅스) : :(콜론)

  - 윈도우에서 사용하는 이름 구분자 : \(역슬래시)
  - 유닉스(리눅스) : /(슬래시)

  - 주요 메소드
   1)파일 정보를 추출
    getName():파일명
    getAbsolutePath(): 경로포함 파일명
    getParent(): 부모 폴더
    length() : 파일 크기
    lastModified() : 최종수정일자 long값으로 나옴
    File.separator(File.separatorChar) : 운영체제에 맞춰 구분자 생성

   2)파일 / 디렉토리 생성
    mkdir() : 만들고자 하는 디렉토리의 상위 디렉토리가
     존재하지 않을 경우 생성 불가

    mkdirs() : 만들고자 하는 디렉토리의 상위 디렉토리가
     존재하지 않을 경우 상위 디렉토리까지 생성

    createNewFile() : txt 파일 생성

   3) 디렉토리 정보 추출
    isDirectory() : true 일 때 존재함
